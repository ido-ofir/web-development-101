# web-development-101

### The most basic HTML page
```html
<!DOCTYPE html>
<html>
  <head>
    
  </head>
  <body>

  </body>
</html>
```

### With a style tag 
```html
<!DOCTYPE html>
<html>
  <head>
    <style>
      body{
        background: red;
      }
    </style>
  </head>
  <body>

  </body>
</html>
```

### With a script tag 
```html
<!DOCTYPE html>
<html>
  <head>
    <style>
      body{
        background: red;
      }
    </style>
    <script>
       alert('who are you?')
    </script>
  </head>
  <body>

  </body>
</html>
```
